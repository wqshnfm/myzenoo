## 什么是冻干狗粮?为什么NASA都在用这技术?

**生产过程:**

1. **优质原料采购**
   从意大利农场采购人类食用级别的肉类和蔬菜。不含激素、抗生素,只用纯肉,不用边角料或副产品。

2. **冻干技术处理**
   食物被冷冻到零下40摄氏度,然后在特殊真空环境下去除水分。这种温和的过程保留了所有珍贵的营养、维生素和矿物质——不像传统热加工那样。

3. **保留天然风味**
   多亏了冻干技术,不仅营养价值得以保留,食材的天然味道和香气也保留了下来。狗狗会得到浓缩但天然的食物。

4. **方便食用**
   只需加温水等几分钟,食物就会恢复原有的质地,可以食用了。干燥形式在室温下保持新鲜,不需要冷藏。

整个生产过程符合欧盟FEDIAF标准,由认证兽医营养师监督,确保100%安全透明。
